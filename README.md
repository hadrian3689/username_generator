# Project Title

Username Generator in C

## Description

A username generator written in C

### Executing program

* First compile
```
gcc username_generator.c -o unamegen
```
* Then execute
```
./unamegen firstname lastname #oflines
```

## Example

* A quick example
```
./unamegen doom guy 100
```
